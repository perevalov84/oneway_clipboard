## Описание
Реализация одностороннего "буфера обмена" мжеду rdp клиентом и сервером
при отключеном буфере обмена на сервере политиками GPO.

Передача буфера обмена организована следующим образом:
- На третьем linux сервере устанавливается oneway-clipboard сервис
- Клиент через powershell скрипт отправляет серверу файлы и содерижмое буфера обмена
- Сервис согласно настройкам data.json определяет отправителя по исходящему ip и
передаёт в сессию пользователя на RDP сервере используя и copyq.
